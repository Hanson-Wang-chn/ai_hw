# “人工智能导论”课程实验作业

## 任务

- 请你帮我编写“人工智能导论”课程实践作业四的代码，分别完成 Seq2Seq.md 中“详细实验设计”部分里的 7 个实验，具体要求如下：
  - 必须严格按照 Seq2Seq.md 中的要求编写代码，且代码可以正常运行。
  - 代码全部放在 `code/` 文件目录下，每一个实验的代码（共7个）分别放在一个子目录中，即 `code/<task_name>/` 目录。在每一问的代码中，如果涉及可视化，请把图片保存到 `code/<task_name>/img/` 文件目录下。
  - 请在 `code/` 文件目录下提供 `run.py`，用于启动某个实验，比如 `python run.py --task <task_name>`。启动某个实验后，必须能**无人值守地**完整运行整个实验流程（包括使用不同的超参数等），且保存必要的日志文件、实验结果、可视化图片等。多组不同的超参数，放在不同的.yaml文件中，分别读取。
  - 请提供 `requirements.txt` 确定相关依赖，适用于一台安装有单张 NVIDIA GPU的 Ubuntu 22.04 无头服务器。Python 版本 为 3.11；torch 版本使用 2.7.1。应尽可能使用 GPU 来提高计算效率。
  - 请提供 `README.md` 文档，包含项目结构、运行方法，并分别对每一个实验的内容、代码、结果保存等做详细说明。注意在中文段落中使用中文逗号“，”，而不是英文逗号“,”。
  - 代码中应包含适量的中文注释。
  - 每一个实验的 VRAM 占用不得超过 14GB。
  - 只需编写正确代码即可，无需立即运行代码。
